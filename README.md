# Covid-19--Power-BI-Dashboard
This project demonstrates the development of an interactive Power BI dashboard for business performance analysis. The dashboard was created to present key metrics, identify trends, compare performance across categories or segments, and communicate insights clearly to business users.
## Objective

The main objective of this project was to transform raw data into a professional and easy-to-understand dashboard that supports business decision-making.

## My Contribution

- Cleaned and transformed data using Power Query.
- Designed the data model for analysis.
- Created DAX measures for KPIs and calculations.
- Built interactive visuals, filters, and slicers.
- Designed a dashboard layout suitable for business stakeholders.
- Presented insights in a clear, practical, and decision-focused way.
